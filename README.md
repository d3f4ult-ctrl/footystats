# Footy Stats
A website that has footballers nd their stats. Purely in Vanilla JS, CSS and HTML only

## Tech Stack
- Vanilla Js - brain
- HTML5 - templates(player's pages)
- CSS3 - styling
- Git - Version control

## Live Demo
[Live Demo](https://d3f4ult-ctrl.github.io/footystats/)

## Features
- Player's Name.
- Player's Nationality
- Player's Age automatically calculated by JavaScript.
- Player's Position(s).
- Player's Picture.
- Player's Goals.
- Player's Assists.
- Player's Total Goal Contributions.
